# Product Requirements v1: Tesseract
> **Status:** Rebranded from FreelancePro CRM

## 🌌 Product Vision
Tesseract is a mystical, AI-powered "Focus Operating System" built for freelancers and solo consultants. It transforms CRM from a data-entry chore into a high-performance cockpit for project orchestration.

## 👥 Target Audience
Elite independent professionals (Designers, Developers, Writers) managing high-value client portfolios (5-50 active projects).

## 💎 Core Features
1. **The Ether (Global Search)**: Cmd+K quick-action palette for context switching.
2. **The Nexus (Dashboard)**: Real-time metrics, agenda, and activity feeds in a unified view.
3. **Chronos (Client Directory)**: Deep-dive timelines and financial health tracking.
4. **Flux (Pipeline)**: Stage-based Kanban movement for active projects.
5. **The Sentinel (AI Support)**: Copilot assistant for intelligent scanning, drafting, and sentiment analysis.

## 🔐 Access & Environments
Tesseract utilizes a multi-context entry flow to ensure data integrity:
- **Phase 0: Demo Mode**: Instant access with "The Void" mock data.
- **Phase 1: Admin Flow**: Secure entry for persistent usage. Requires PIN `1510`.
- **Phase 2: Test / UAT**: Isolated sandbox for testing new features. Requires PIN `0000`. Includes "Hard Reset" capabilities.

## 🎯 Success Metric
Seamless transition between data contexts with zero cross-contamination, maintaining a "Pitch Black" focus and high-speed interaction.
