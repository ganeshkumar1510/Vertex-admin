# Tesseract Vertex: The Focus Operating System

> **A premium, multi-mode CRM for craftsmen who value mental clarity and high-performance focus.**

![Tesseract Concept](https://images.unsplash.com/photo-1614850523296-d8c1af93d400?q=80&w=2070&auto=format&fit=crop)

## 🌌 The Vision
Tesseract Vertex is more than a CRM—it's a **private operating system for craft**. Built for independent craftsmen—developers, designers, and consultants—who prioritize deep work and high-quality client relationships over raw volume.

This version (Vertex) introduces a sophisticated dual-mode architecture:
- **Aether Mode**: An immersive, cosmic violet dark mode inspired by the depth of space.
- **Quasar Mode**: A clinical, high-contrast monochrome light mode for zero-distraction productivity.

## 🚀 Getting Started

### 🛠️ Prerequisites
- Node.js (v18+)
- npm or yarn

### 📥 Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/tesseract-vertex.git
    cd tesseract-vertex
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Environment Setup**:
    - Copy `.env.example` to `.env`
    - Define your security sequences: `VITE_ADMIN_PIN` and `VITE_TEST_PIN`.
4.  **Launch the Engine**:
    ```bash
    npm run dev
    ```

## 🎨 The Vertex Design System
Tesseract Vertex follows a strict architectural design language:
- **Dynamic Theming**: Seamlessly toggle between Aether and Quasar modes.
- **Obsidian Aesthetic**: Pitch-black canvas, 1px border geometry, and zero unnecessary shadows.
- **Typography**: Industrial legibility using **Instrument Sans** and **JetBrains Mono**.

Detailed specifications are available in [docs/design_system.md](./docs/design_system.md).
